# electron-memo
A tiny memo based on Electron8 and Vue

使用vue + electron + nodejs写的便签应用

[ ] nodejs写入本地数据库

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
